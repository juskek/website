### Licks / Runs

#### Oscar Peterson - Body and Soul

Built from notes in F7b9 + Fdim7 + F#dim7. Play F7b9 in left hand.

```lilypond
{
  \relative c' {
    f8 fis a bes
    \tuplet 3/4 {c des ees}
  }
  \relative c' {
    f8 fis a bes
    \tuplet 3/4 {c des e}
  }
  \relative c' {
    f8 fis a bes
    c des ees e
  }
}
```
